#### Create a Node.js project and publish it on Github

This [Introduction to NPM and Node](https://omundy.github.io/learn-javascript/topics/node/slides.html) tutorial builds a foundation for what you will learn in this class. Below find the learning objectives for this DevLog which correspond to sections in the tutorial. Save the information with the ✏️ which you will enter in your DevLog in the next class. <IconTime>:40 min</IconTime>

1. Install and use Node.js on your machine
1. Confirm your Node version ✏️`v25.4.0`
1. Run a script with Node.js
1. Initialize a new project with NPM
1. Install Nodemon globally to automatically reload scripts
1. Install the Cron NPM package and use it in a script
1. [Look up](https://crontab.guru) the cron expression for "every 10 minutes". Test it and add it here: '10,20,30,40,59 0 * * *' {/* https://crontab.guru/every-10-minutes */}
1. Create a Git repo and commit a `.gitignore` file
1. Publish the repo on Github. Add the link [here] (https://github.com/siqjulia/cron-demo.git)
1. Turn it in by posting the Github link in Moodle
